Event Binding

You can find the code for this portion in src/app/components/event-binding, where you should only be modifying the event-binding.component.html and event-binding.component.ts files. In the component's HTML code, the binding (click)="processClickButton1() has been added to the button 1 element. That means that when the (click) event fires, the processClickButton1 method in the Component's TypeScript file will run. In this case, property binding is later used to helpfully inform the user that the button has been clicked.
